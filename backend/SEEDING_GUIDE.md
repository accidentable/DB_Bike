# 데이터베이스 시딩 스크립트

## 변경 사항

모든 분산된 seed 파일들을 **`seedDatabase.js` 하나로 통합**했습니다.

### 통합된 파일
- **`seedDatabase.js`** - 모든 작업을 한 번에 수행

---

## 사용 방법

```bash
npm run db:seed
```

이 명령어 하나로 다음을 모두 자동으로 수행합니다:

1. 📝 **데이터베이스 초기화** (init.sql 실행)
   - 테이블 생성
   - 회원 데이터 삽입 (admin, user 계정)

2. 🚴 **서울시 공공자전거 대여소 정보** (API 호출)
   - 2,760개 대여소 추가
   - 36,501개 자전거 생성

3. 🏆 **랭킹 데이터 (대여 기록)**
   - 10명의 랭킹 사용자 생성
   - 201개 대여 기록 생성 (거리 및 시간 랜덤)

4. 💬 **커뮤니티 게시글 목록** (목업 데이터)
   - 공지사항 3개 (고정)
   - 이벤트 2개 (고정)
   - 자유 게시판 8개
   - 후기 게시판 10개
   - 제안 게시판 10개
   - **총 33개 게시글**

---

## 특징

✨ **공존 가능**: 모든 데이터가 일관성있게 생성되므로 공존 가능

⚡ **빠른 실행**: 병렬 처리와 최적화로 빠른 성능 제공

📋 **상세한 로그**: 각 단계별 진행 상황을 명확히 표시

---

## 주의사항

⚠️ **기존 데이터 삭제**: `seedDatabase.js` 실행 시 모든 기존 데이터가 삭제됩니다.

🔑 **API 키 필수**: `.env` 파일에 `SEOUL_API_KEY` 설정이 필요합니다.
